The best road-trip country in America starts here. Answer a few questions, click what appeals, and build a trip you can print and book.

To embed this on trlibrary.com (Drupal): paste the two lines below into a Full-HTML block or Custom-HTML field.

<div id="tr-trip-planner"></div>
<script src="https://trip.labs.trlibrary.com/assets/trip-planner.js" data-container="tr-trip-planner"></script>

Analytics (optional): if the host page already has Google Analytics, the planner forwards anonymous usage events automatically. To send them to a specific GA4 property instead, add data-ga="G-XXXXXXXXXX" to the script tag.
